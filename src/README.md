# Python Desktop-App (Arbeitsnachweis)

Dieses Projekt ist die Desktop-Version, programmiert in Python.

## Start

```powershell
python src/ruf_arbeitsnachweis.py
```

## Inhalt

- `src/ruf_arbeitsnachweis.py`: Hauptanwendung
- `src/Arbeitsnachweis_Modern.pdf`: Vorlage/Beispieldokument
- `src/Arbeitsnachweis_Draft.txt`: Notizen/Entwurf
- `src/signature_tech.png`: Grafikressource

## Hinweis

Die Web-App ist ein separates Projekt im Ordner `web_arbeitsnachweis/`.
